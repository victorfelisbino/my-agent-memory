# Security Policy

## Supported Versions

This framework is maintained on the latest main branch.

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities.

Report privately using one of these paths:
- GitHub Security Advisories for this repository (preferred)
- Direct maintainer contact if advisories are unavailable

Include:
- Impact summary
- Reproduction steps
- Affected files or scripts
- Suggested fix if known

You can expect an initial response within 5 business days.

## Sensitive Data Guidance

This repository is framework-only. Never include personal memory files, transcripts, credentials, tokens, or private URLs in issues, pull requests, or logs.
