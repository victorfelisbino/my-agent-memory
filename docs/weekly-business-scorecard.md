Use this once per week to convert business rules into measurable decisions.

## How to use

1. Fill this for the last completed week (Mon-Sun or your operating cadence).
2. Mark each KPI as Green, Yellow, or Red against the default threshold.
3. Pick exactly 2 interventions for next week.
4. Log one counter-opinion before locking the plan.

## Scorecard template

Week of: __________

1. Adjacent revenue share (multi-profit-center health)
- KPI: % of revenue from adjacent offers (not core offer).
- Formula: adjacent revenue / total revenue.
- Default threshold: Green >= 30%, Yellow 20-29%, Red < 20%.
- Current: ________
- Status: [ ] Green [ ] Yellow [ ] Red

2. Service reliability index (trust moat)
- KPI: weighted reliability score across uptime, response speed, and failure rate.
- Formula: 0.4 * uptime% + 0.3 * on-time response% + 0.3 * (100 - failure%).
- Default threshold: Green >= 95, Yellow 90-94.9, Red < 90.
- Current: ________
- Status: [ ] Green [ ] Yellow [ ] Red

3. Expansion cadence execution (repeatable growth)
- KPI: planned openings/upgrades completed this week.
- Formula: completed / planned.
- Default threshold: Green >= 90%, Yellow 75-89%, Red < 75%.

4. Retention loop engagement (durable demand)
- KPI: % of active users/accounts using at least one retention loop (loyalty, recurring service, account program).
- Formula: engaged active users / total active users.
- Default threshold: Green >= 60%, Yellow 45-59%, Red < 45%.

5. Workforce readiness (execution capacity)
- KPI: % of critical roles/skills meeting readiness standard.
- Formula: roles ready / critical roles.
- Default threshold: Green >= 90%, Yellow 80-89%, Red < 80%.

6. Revenue mix resilience (concentration risk)

## Competitive watch (weekly)

Track these as trend signals versus last week.

1. Bundle velocity gap
- KPI: competitor bundle launches this month - our bundle launches this month.
- Default threshold: Green <= 0, Yellow 1-2, Red >= 3.
- Current: ________
- Trend: [ ] Improving [ ] Flat [ ] Worsening

2. Loyalty engagement trend
- KPI: % point change in active loyalty users (our segment) vs market benchmark.
- Default threshold: Green >= +2 pp, Yellow 0 to +1.9 pp, Red < 0 pp.
- Current: ________
- Trend: [ ] Improving [ ] Flat [ ] Worsening

3. Service reliability gap
- KPI: our service reliability index - best competitor service reliability index.
- Default threshold: Green >= 0, Yellow -0.1 to -2.0, Red < -2.0.
- Current: ________
- Trend: [ ] Improving [ ] Flat [ ] Worsening

4. Fleet-finance attachment rate
- KPI: % of fleet accounts using at least one financing/billing product.
- Default threshold: Green >= 40%, Yellow 25-39%, Red < 25%.
- Current: ________
- Trend: [ ] Improving [ ] Flat [ ] Worsening

5. Promo intensity risk
- KPI: competitor discount/promotional actions affecting our core routes this week.
- Default threshold: Green <= 2, Yellow 3-5, Red >= 6.

- Trend: [ ] Improving [ ] Flat [ ] Worsening

## Rules of use

!!! warning "Decision hygiene"
    - Do not change thresholds and interventions in the same week unless there is a clear data-quality issue.
    - If 2+ KPIs are Red for 2 consecutive weeks, freeze new expansion and run a focused reliability sprint.
    - Revisit thresholds monthly based on outcomes; treat these as defaults, not doctrine.
