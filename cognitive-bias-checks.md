# Cognitive Bias Checks

Use this checklist to avoid bad judgment loops.

## Common traps

- Confirmation bias: only looking for evidence that supports current belief.
- Recency bias: overweighting the latest incident.
- Authority bias: accepting claims without verification because source sounds strong.
- Action bias: doing more instead of thinking better.
- Sunk cost fallacy: continuing because we already invested effort.

## Debiasing prompts

- What evidence would prove me wrong?
- What is the strongest alternative explanation?
- If I started fresh, would I choose the same path?
- What would I advise someone else to do here?
- Which assumption is least tested?

## Decision hygiene

- Write one sentence for "why now".
- Write one sentence for "what could fail".
- Define a stop condition before execution.
